Have a different static frontpage for logged in visitors

== Description ==

Do you use a static front page but want to display different information for you members. Well not you can. This plugin allows you too set a different front page for logged in users. 

It is very simple to set up and just works.


== Installation ==

1. Upload the entire `lh-logged-in-static-frontpage` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Navigate to Settings->Reading to set the page ID of the page for logged in visitors on the frontend.
